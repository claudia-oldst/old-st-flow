# Public client portal — Timeline tab

Default tab of the public client portal.

## Content
- A read-only Sprint Gantt for the project: epics on the y-axis, sprints on the x-axis.
- Each cell shows progress (committed → done) as a segmented bar.
- A "Today" marker is drawn as a dashed vertical line.

## Interactions
- Hovering a cell shows a tooltip with the epic name, sprint name, and progress numbers.
- No editing or drag interactions — purely informational.

## Empty state
"No sprints yet" friendly message when the project has no roadmap configured.
